# Vome 插件规范（纯前端 · vome-plugin-front）

> 目录：`plugins/vome-plugin-front` · key：`scaffold-frontend`

## 可用 / 不可用

- **可用**：Vue 微应用、menus/wujie、`hostRequest`（Bearer + `/dev` 前缀）  
- **不推荐**：整包 admin CRUD  
- **禁止**：声明 hook/routes 无 server；`base: '/'`

## 产物

`pack` → 含 `web/` + `locales/`；源码在 `src/`，用 `src/lib/host-api.ts` 调宿主。

## 多语言

- **源包**：包根 `locales/zh-CN.json`（`pack` 必须打进 `.vome`，安装后宿主「同步语言包」可扫到）
- **译包权威**：宿主表 `i18n_pack`（`scopeType=plugin` / `scopeKey=module.json.key`）；插件勿另起一套本地 `en-US.json` 等
- **运行时**：`src/lib/locale.ts` 的 `usePluginLocale()` — `zh-CN` 用本地源 JSON，其它语种 `GET /admin/i18n/pack/active`；语种列表 `GET /admin/i18n/lang/enabled`（含 `flag`）
- **UI**：右上角国旗按钮 + 下拉（左国旗、右语种名）；示例文案一律 `t('a.b')`
